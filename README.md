# 项目日报管理系统

---
在工作之余，做的日报管理系统，用于每天写日报，并且具有文档导出功能

项目开发：Java，Mybatis，Maven，Spring，前端用的layui

运行环境：

jdk:1.8.0_201

mysql:5.6.35

数据库存放位置：daily\src\main\resources\daily_report.sql
